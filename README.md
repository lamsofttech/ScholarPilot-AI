# PhD Funding Agent

PhD Funding Agent is an agentic AI system designed to help individuals discover, match with, and win fully funded PhD scholarships.

## Objective

The main objective of this project is to build an AI-powered system that supports a PhD applicant throughout the scholarship application journey.

The system will help users:

- Build a clear academic and research profile
- Discover fully funded PhD opportunities
- Match their profile with suitable scholarships and universities
- Understand eligibility requirements
- Prepare strong application materials
- Draft supervisor outreach emails
- Improve statements of purpose and research proposals
- Track deadlines, documents, and application progress
- Increase their chances of winning PhD funding

## Vision

The platform should work like a personal PhD funding assistant. Instead of only listing scholarships, it should guide the applicant step by step from searching for opportunities to submitting strong applications.

## Core Features

### 1. Applicant Profile Agent

Collects and organizes the user's academic background, research interests, target countries, preferred programs, funding needs, and application readiness.

### 2. Funding Research Agent

Searches for PhD scholarships, funded research positions, assistantships, fellowships, and university-based funding opportunities.

### 3. Scholarship Matching Agent

Matches the applicant's profile with relevant funding opportunities and explains why each opportunity is a good or weak fit.

### 4. Application Strategy Agent

Creates a step-by-step application plan based on deadlines, requirements, and applicant readiness.

### 5. Document Assistant Agent

Helps prepare and improve documents such as:

- Academic CV
- Statement of Purpose
- Research Proposal
- Personal Statement
- Scholarship Essays
- Supervisor Outreach Emails
- Recommendation Request Emails

### 6. Deadline Tracker Agent

Tracks scholarship deadlines, document status, supervisor responses, and submitted applications.

## Technology Direction

The system may include:

- Frontend: React and TypeScript
- Backend: PHP APIs
- Database: MySQL or MariaDB
- AI Layer: LLM-powered agent workflows
- Future Integrations: Web search, email, calendar, document parsing, and scholarship databases

## Long-Term Goal

The long-term goal is to create an intelligent scholarship success platform that gives applicants a structured, personalized, and strategic path toward winning fully funded PhD opportunities.
